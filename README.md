# MyCam

将Android手机摄像头作为PC虚拟摄像头使用的应用。

## 功能特性

- 📱 将手机摄像头转换为PC虚拟摄像头
- 🌐 通过HTTP MJPEG流传输视频
- 🔄 支持前后摄像头切换
- ⚡ 优化的视频编码，支持高帧率
- 🎨 现代化Material Design 3界面

## 系统要求

- Android 12+ (API 31+)
- 支持摄像头功能
- WiFi网络连接

## 安装说明

### 方法1：直接安装APK
1. 下载 `app-release.apk` 文件
2. 在手机上启用"未知来源"安装
3. 点击APK文件进行安装

### 方法2：通过ADB安装
```bash
adb install app-release.apk
```

## 使用说明

1. **启动应用**：点击应用图标启动MyCam
2. **授权权限**：允许摄像头和通知权限
3. **开始直播**：点击"Start"按钮开始视频流
4. **获取地址**：复制显示的IP地址和端口
5. **PC端使用**：在PC软件中输入 `http://手机IP:8080/mjpeg`

## PC端配置

### OBS Studio
1. 添加"媒体源"
2. 输入URL：`http://手机IP:8080/mjpeg`
3. 选择"本地文件"取消勾选
4. 点击"确定"

### VLC播放器
1. 打开VLC
2. 媒体 → 打开网络串流
3. 输入URL：`http://手机IP:8080/mjpeg`
4. 点击播放

## 故障排除

### 无法连接
- 确保手机和PC在同一WiFi网络
- 检查防火墙设置
- 确认手机IP地址正确

### 视频卡顿
- 降低JPEG质量设置
- 选择较低分辨率
- 确保网络信号良好

### 权限问题
- 重新授权摄像头权限
- 检查通知权限设置

## 技术规格

- **视频格式**：MJPEG
- **默认端口**：8080
- **支持分辨率**：480p, 720p, 1080p
- **帧率**：最高30fps
- **编码**：JPEG压缩

## 隐私说明

- 本应用不会收集任何个人数据
- 视频流仅在本地网络传输
- 不会上传到任何服务器
